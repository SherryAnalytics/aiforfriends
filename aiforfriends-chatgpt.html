ChatGPT 是 OpenAI 公司开发的 AI 助手，也是目前全球最多人使用的 AI 工具。 它特别擅长生成图片、语音对话和联网搜索。打开 chatgpt.com，免费注册即可开始。
入门 Level 1
第一次对话Say Hello
  • 1打开 chatgpt.com，注册账号
  • 2在输入框用中文或英文打一句话
  • 3按 Enter，读回答
  • 4不满意？直接说"再解释得简单一点"
试试说 / Try saying
"今晚想做红烧肉，帮我写出步骤，我是新手。"
基础 Level 2
翻译与改写Translate & Rewrite
  • 1直接把要翻译的文字粘贴进去
  • 2说明目标语言和用途
  • 3要求调整语气：正式 / 口语 / 亲切
  • 4让它帮你润色、简化或扩展内容
试试说 / Try saying
"帮我把这封英文邮件翻译成中文，语气要自然一点，不要太生硬。"
进阶 Level 3
用语音聊天Voice Chat
  • 1手机 App 下载 ChatGPT
  • 2点击麦克风图标开始说话
  • 3用中文或英文对话，它都能听懂
  • 4适合练习英文口语或免打字场景
试试说 / Try saying
（用语音）"帮我练习一段英文自我介绍，然后给我纠正发音建议。"
中级 Level 4
生成和修改图片Create & Edit Images
  • 1用文字描述你想要的图片
  • 2上传已有照片，要求修改或美化
  • 3可以换背景、加文字、改风格
  • 4生成后觉得不对，继续描述调整
试试说 / Try saying
"帮我把这张照片的背景换成樱花公园，保持人物不变。"
"画一张温馨的中式家庭聚餐插图，卡通风格。"
高级 Level 5
联网搜索与查询Search the Web
  • 1开启 Search 功能（付费版）
  • 2问最新新闻、价格、活动信息
  • 3让它比较不同产品或服务
  • 4它会引用来源，方便核实
试试说 / Try saying
"帮我查一下凤凰城本周末有什么华人文化活动，列出地点和时间。"
精通 Level 6
自定义 GPTCustom GPTs
  • 1在 GPT Store 找到专用 AI 工具
  • 2或自己创建一个专属 GPT
  • 3上传你的资料，训练专属助手
  • 4分享给家人或团队一起用
试试说 / Try saying
"帮我创建一个专门回答家庭理财问题的 GPT，用中文回答，语气像朋友一样。"
