Claude 是 Anthropic 公司开发的 AI 助手。只需要用自然语言和它说话，就像和朋友聊天一样。 无需编程，无需技术背景。打开 claude.ai，免费注册即可开始。
入门 Level 1
第一次对话Say Hello
  • 1打开 claude.ai，注册账号
  • 2在输入框用中文或英文打一句话
  • 3按 Enter，读回答
  • 4觉得不满意？直接说"再简单一点"
试试说 / Try saying
"今晚冰箱里有鸡蛋、豆腐、青菜，帮我想三个快手晚餐。"
基础 Level 2
给它更多背景Add Context
  • 1告诉 Claude 你是谁、做什么的
  • 2说清楚你想要什么结果
  • 3指定语气：正式 / 轻松 / 简短
  • 4不满意就继续对话，不用重新开始
试试说 / Try saying
"帮我把这段房东通知翻译成中文，再解释一下我需要做什么。"
进阶 Level 3
让它扮演角色Assign a Role
  • 1告诉 Claude 扮演某个专家
  • 2描述你的具体场景
  • 3要求它按步骤解释
  • 4追问细节，深入对话
试试说 / Try saying
"我们一家四口要去日本京都玩五天，预算适中，帮我规划一个行程，包括吃饭和交通。"
中级 Level 4
处理文件和数据Work with Files
  • 1上传 PDF、Excel、图片文件
  • 2让 Claude 总结、翻译或分析内容
  • 3要求它提取关键数字或信息
  • 4让它帮你写回复邮件或报告
试试说 / Try saying
"这是一张我拍的收据，帮我列出所有消费项目和总金额。"
"帮我把这封英文信改得更礼貌正式一点。"
高级 Level 5
建立工作流程Build a Workflow
  • 1创建 Project（项目），保存上下文
  • 2把常用 Prompt（提示词）保存复用
  • 3让 Claude 生成模板，定期更新
  • 4把 AI 融入你的日常工作节奏
试试说 / Try saying
"帮我设计一周的家庭晚餐菜单，要健康少油，顺便列出购物清单。"
精通 Level 6
个人知识与研究Personal Projects
  • 1上传笔记、文章、家史资料
  • 2让 Claude 整理成知识库或时间线
  • 3提问、对比、生成报告
  • 4结合网络搜索做深度研究
试试说 / Try saying
"这些是我爷爷留下的老照片和信件，帮我整理成家族故事，用讲故事的方式写出来。"

💡 好 Prompt 的秘诀 · Better Prompts

  • 说清楚你是谁（身份背景）
  • 说清楚你想要什么（目标）
  • 指定格式：列表、段落、表格
  • 不满意就继续说，不要重新来过
  • 可以用中文、英文，或两者混用

⚠️ 记住这几点 · Keep in Mind

  • Claude 不联网，信息截止到训练数据日期
  • 重要决策请自行核实（法律、医疗、财务）
  • 不要输入密码、银行账号等隐私信息
  • 免费版已经很强大，够日常使用
  • 多用多熟悉，像学用一个新工具
